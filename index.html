  • One Time Pad Cipher

    A one-time pad (OTP) cipher is an encryption algorithm that uses a secret key of the same length as the message being encrypted. The key is generated randomly and is used only once for a single message. The security of the one-time pad lies in the fact that if the key is truly random, used only once, and kept secret, it provides perfect secrecy.
